O método fromEvent do RxJS é utilizado para criar um Observable a partir de um evento DOM ou de outro tipo de evento. Ele aceita dois argumentos: o primeiro é a origem do evento (um elemento DOM, um objeto EventEmitter, etc.) e o segundo é o nome do evento que queremos observar.

O Observable criado por fromEvent continua emitindo novos eventos sempre que eles acontecem na origem (no caso, o botão). É importante lembrar de cancelar a inscrição (usando o método unsubscribe()) quando não precisar mais receber esses eventos, para evitar vazamentos de memória.

O método fromEvent também pode ser usado para observar eventos de outros tipos de objetos que emitem eventos, como o WebSocket. Em resumo, o método fromEvent é útil para transformar eventos em Observable de forma simples e fácil de usar.
